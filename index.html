Cleaning pipeline

11-step pipeline. Zero manual work.

01
Remove empty rows & null columns
Drops fully empty rows and removes columns with no useful data.
02
Normalize column names
Removes spaces, symbols and fixes naming consistency.
03
Replace placeholders & fix encoding
Converts "n/a", "null", "unknown" into proper NaNs and fixes encoding.
04
Smart type casting
Automatically converts columns to correct numeric, categorical, or datetime formats.
05
Drop high-missing columns
Removes columns with more than 55% missing values.
06
Fill missing values
Numeric → median, categorical → mode.
07
Remove duplicate rows
Ensures dataset uniqueness.
08
Standardize text & categories
Fixes casing, trims whitespace, and cleans categorical values.
09
Winsorize outliers
Caps extreme values to reduce skewness without removing data.
10
Fix impossible values
Corrects unrealistic values like negatives or invalid ranges.
11
Remove constant & correlated features
Drops redundant and highly correlated columns.

Dataset Quality Score

A score for every dimension of data health.

DQS is a weighted composite across 7 dimensions — completeness, uniqueness, consistency, validity, accuracy, structure, and correlation. Adjusted for dataset size and type diversity.
