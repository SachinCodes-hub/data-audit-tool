# 🚀 Data Audit Tool

A premium, web-based data analysis and cleaning platform that evaluates dataset quality and transforms raw data into actionable insights.

---

## ✨ Features

* 📊 **Data Quality Score (DQS)**

  * Completeness, consistency, validity, uniqueness
  * Outlier detection (Z-score & IQR)
  * Correlation & redundancy analysis

* 🧹 **Advanced Data Cleaning Pipeline**

  * Missing value handling (mean, median, KNN)
  * Duplicate removal (exact + fuzzy)
  * Outlier treatment
  * Feature scaling & encoding
  * Feature engineering & selection
  * Data balancing & transformation

* 🔍 **Deep Insights**

  * Correlation heatmaps
  * Distribution analysis
  * Skewness & feature importance
  * Pattern & anomaly detection

* 🔁 **Before vs After Comparison**

  * Data Quality Score improvement
  * Visual comparison of dataset

* 💎 **Premium UI**

  * Dark luxury theme
  * Glassmorphism + neon glow design
  * Smooth animations
